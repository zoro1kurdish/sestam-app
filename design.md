# Accounting System Design Style

## Design Philosophy

### Color Palette
- **Primary Blue**: #2563eb (Professional trust and reliability)
- **Secondary Gray**: #64748b (Neutral business tone)
- **Accent Green**: #059669 (Success and positive financial data)
- **Warning Orange**: #ea580c (Alerts and important notifications)
- **Background**: #f8fafc (Clean, minimal background)
- **Text Dark**: #1e293b (High contrast readability)

### Typography
- **Display Font**: "Inter" - Modern, professional sans-serif for headings
- **Body Font**: "Inter" - Consistent, readable for all content
- **Arabic Font**: "Cairo" - Professional Arabic typography
- **Kurdish Font**: "Inter" with proper RTL support

### Visual Language
- **Minimalist Business**: Clean, professional interface with ample white space
- **Data-Driven**: Charts and visualizations as primary visual elements
- **Trust & Security**: Subtle shadows and borders conveying stability
- **Accessibility**: High contrast ratios and clear visual hierarchy

## Visual Effects

### Used Libraries
- **ECharts.js**: Interactive financial charts and data visualization
- **Anime.js**: Smooth micro-interactions and form animations
- **Typed.js**: Dynamic text effects for key metrics
- **Splitting.js**: Text animation effects for headings
- **Matter.js**: Physics-based loading animations
- **Pixi.js**: Advanced visual effects for dashboard backgrounds

### Animation Effects
- **Typewriter Animation**: Key financial metrics appearing dynamically
- **Chart Animations**: Smooth data transitions and updates
- **Form Micro-interactions**: Subtle feedback on user actions
- **Loading States**: Professional loading animations for data processing
- **Hover Effects**: Gentle elevation and color transitions

### Header Effect
- **Animated Background**: Subtle geometric patterns using Pixi.js
- **Dynamic Metrics**: Real-time financial KPIs with typewriter effect
- **Gradient Overlays**: Professional color transitions
- **Responsive Design**: Adapts to different screen sizes

### Styling Approach
- **Card-Based Layout**: Organized information in clean, bordered cards
- **Consistent Spacing**: 8px grid system for perfect alignment
- **Subtle Shadows**: Depth without distraction
- **Professional Icons**: Consistent iconography throughout
- **Responsive Grid**: Flexible layout for all devices

### RTL Support
- **Arabic/Kurdish Layout**: Full right-to-left interface support
- **Mirrored Components**: Properly aligned forms and navigation
- **Font Optimization**: Appropriate typography for Arabic script
- **Cultural Sensitivity**: Color and layout considerations for target markets
